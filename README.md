## 📝 Описание

**Racing Game 3D** — это гоночная игра с видом от третьего лица, где два игрока соревнуются на одной трассе.  
Игра работает в реальном времени через WebSockets и поддерживает локальный мультиплеер.

### Особенности:
- 🏁 **3D-графика** – на основе Three.js
- 🔧 **Физика автомобилей** – Cannon.js (реалистичное управление, столкновения)
- 👥 **Мультиплеер** – до 2 игроков через локальную сеть или интернет
- 💬 **Чат** – общайтесь с соперником во время гонки
- 📊 **База данных** – PostgreSQL с Prisma ORM
- 🏆 **Система очков** – за опасные манёвры и обгоны
- 🚧 **Трафик** – препятствия на дороге
- 🎮 **Управление** – W/A/S/D, пробел для экстренного торможения

---

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| **W** | Ускорение |
| **S** | Торможение / задний ход |
| **A** | Поворот налево |
| **D** | Поворот направо |
| **Пробел** | Экстренное торможение |
